Constructor detail
FPolyFromPoints()constructor
public function FPolyFromPoints($gfx:Graphics, $points:Array, $fill:IFill = null, $autoRedraw:Boolean = true)

Constructs and draws a polygon from a compound array.

Parameters
$gfx:Graphics — Reference to the shape.graphics where the shape will be drawn into.
 
$points:Array — Compound array of x and y coordinate points.
 
$fill:IFill (default = null) — Fill or line style to apply the shape.
 
$autoRedraw:Boolean (default = true) — Determines if the graphic should auto redraw when a setter is called.

Example
The following code creates a polygon shape. 
    var sf :SolidFill = new SolidFill( 0x666666, .8 );
    var p1 :FPolyFromPoints = new FPolyFromPoints( _gfx.graphics, [ [0, 0], [0, 100], [100, 200], [ 350, 300] ], sf );
