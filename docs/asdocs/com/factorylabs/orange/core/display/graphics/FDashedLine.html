Constructor detail
FDashedLine()constructor
public function FDashedLine($gfx:Graphics, $x:Number = 0, $y:Number = 0, $endX:Number = 10, $endY:Number = 0, $dashLength:Number = 1, $gapLength:Number = 1, $fill:IFill = null, $autoRedraw:Boolean = true)

Constructs and draws a new dashed line.

Parameters
$gfx:Graphics — Reference to the shape.graphics where the shape will be drawn into.
 
$x:Number (default = 0) — Start x coordinate value to draw the shape.
 
$y:Number (default = 0) — Start y coordinate value to draw the shape.
 
$endX:Number (default = 10) — End x coordinate value to draw the shape.
 
$endY:Number (default = 0) — End y coordinate value to draw the shape.
 
$dashLength:Number (default = 1) — Length of each dash segment.
 
$gapLength:Number (default = 1) — Length of space between each dash segment.
 
$fill:IFill (default = null) — Fill or line style to apply the shape.
 
$autoRedraw:Boolean (default = true) — Determines if the graphic should auto redraw when a setter is called.

Example
The following code creates a dashed line. 
    var ss :SolidStroke = new SolidStroke( 0xcccccc, 1, 3 );
    var dl :FDashedLine = new FDashedLine( _gfx.graphics, 0, 0, 500, 0, 5, 5, ss );
