Constructor detail
FPolygon()constructor
public function FPolygon($gfx:Graphics, $x:Number = 0, $y:Number = 0, $sides:uint = 3, $radius:Number = 5, $angle:Number = 0, $fill:IFill = null, $center:Boolean = false, $autoRedraw:Boolean = true)

Constructs and draws a new polygon with a minimum of 3 sides.

Parameters
$gfx:Graphics — Reference to the shape.graphics where the shape will be drawn into.
 
$x:Number (default = 0) — A number indicating the horizontal position relative to the registration point of the parent display object.
 
$y:Number (default = 0) — A number indicating the vertical position relative to the registration point of the parent display object.
 
$sides:uint (default = 3) — The number of sides for the polygon.
 
$radius:Number (default = 5) — The of the polygon to draw.
 
$angle:Number (default = 0) — Rotation angle of the polygon. Valid values are between 0 and 360.
 
$fill:IFill (default = null) — Fill or line style to apply to the shape.
 
$center:Boolean (default = false) — Whether to draw the shape from a center registration.
 
$autoRedraw:Boolean (default = true) — Determines if the graphic should auto redraw when a setter is called.

Example
The following code creates a 5 sided polygon. 
    var sf :SolidFill = new SolidFill( 0x666666, .8 );
    var p1 :FPolygon = new FPolygon( _gfx.graphics, 50, 0, 5, 50, 0, sf );
