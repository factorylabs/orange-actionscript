Classpublic class Graphic
ImplementsIGraphic
SubclassesFDashedLine, FEllipse, FPolyFromPoints, FPolygon, FRectangle, FRectRound, FRectRoundComplex, FWedge

Graphic handles the basic necessities for creating a shape and starting and ending a fill or stroke through oranges drawing API.

This is an abstract base class and is never instantiated. It merely stores common properties and internal routines for all shape classes.

Base Constructor for setting and storing instance properties called from a subclass.

Parameters
$gfx:Graphics (default = null) — Reference to the shape.graphics where the shape will be drawn into.
 
$x:Number (default = 0) — A number indicating the horizontal position relative to the registration point of the parent display object.
 
$y:Number (default = 0) — A number indicating the vertical position relative to the registration point of the parent display object.
 
$width:Number (default = 10) — The width of the shape.
 
$height:Number (default = 10) — The height of the shape.
 
$fill:IFill (default = null) — Fill or line style to apply to the shape.
 
$center:Boolean (default = false) — Whether to draw the shape from a center registration.
 
$autoRedraw:Boolean (default = true) — Determines if the graphic should auto redraw when a setter is called.
Method detail
draw()method
protected function draw():void

Calls subroutines for starting a fill/stroke, drawing the shape, and closing the fill/stroke.

This will fail silently if the graphics object is null. If there is a null fill, it will draw the shape but not fill it.

drawGraphic()method 
protected function drawGraphic():void

Handles drawing the shape.

redraw()method 
public function redraw():void

Redraws the shape.

If the autoRedraw property is set to true, a setter will automatically call this method.

Note: This will call clear() on the shape.graphics object and clear all existing grapics witihin that object.
