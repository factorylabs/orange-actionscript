Constructor detail
FRectRoundComplex()constructor
public function FRectRoundComplex($gfx:Graphics, $x:Number = 0, $y:Number = 0, $width:Number = 10, $height:Number = 10, $fill:IFill = null, $tl:Number = 0, $tr:Number = 0, $bl:Number = 0, $br:Number = 0, $center:Boolean = false, $autoRedraw:Boolean = true)

Constructs and draws a rounded rectangle shape with an option to apply different values for individual corners.

Parameters
$gfx:Graphics — Reference to the shape.graphics where the shape will be drawn into.
 
$x:Number (default = 0) — A number indicating the horizontal position relative to the registration point of the parent display object.
 
$y:Number (default = 0) — A number indicating the vertical position relative to the registration point of the parent display object.
 
$width:Number (default = 10) — The width of the shape.
 
$height:Number (default = 10) — The height of the shape.
 
$fill:IFill (default = null) — Fill or line style to apply the shape.
 
$tl:Number (default = 0) — The top left corner value.
 
$tr:Number (default = 0) — The top right corner value.
 
$bl:Number (default = 0) — The bottom left corner value.
 
$br:Number (default = 0) — The bottom right corner value.
 
$center:Boolean (default = false) — Whether to draw the shape from a center registration.
 
$autoRedraw:Boolean (default = true) — Determines if the graphic should auto redraw when a setter is called.

Example
The following code creates a complex rounded rectangle. 
    var sf :SolidFill = new SolidFill( 0x666666, .8 );
    var r1 :FRectRoundComplex = new FRectRoundComplex( _gfx.graphics, 0, 0, 200, 100, sf, 10, 10, 0, 0 );
