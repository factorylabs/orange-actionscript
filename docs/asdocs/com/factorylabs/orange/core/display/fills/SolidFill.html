Constructor detail
SolidFill()constructor
public function SolidFill($color:uint = 0xff00ff, $alpha:Number = 1)

Constructs a new solid fill.

Parameters
$color:uint (default = 0xff00ff) — A hexadecimal color value of the fill.
 
$alpha:Number (default = 1) — A number that indicates the alpha value of the color of the fill; valid values are 0 to 1.

Example
The following code creates a grey solid fill with 80% opacity. 
    var sf :SolidFill = new SolidFill( 0x666666, .8 );
