Constructor detail
BitmapFill()constructor
public function BitmapFill($bitmapData:BitmapData, $source:IBitmapDrawable, $repeat:Boolean = true, $smooth:Boolean = true, $blendMode:String = null, $clipRect:Rectangle = null, $rotation:Number = 0, $colorTransform:ColorTransform = null)

Constructs a new BitmapFill fill.

Parameters
$bitmapData:BitmapData — A transparent or opaque bitmap image that contains the bits to be displayed.
 
$source:IBitmapDrawable — The display object or BitmapData object to draw to the BitmapData object. (The DisplayObject and BitmapData classes implement the IBitmapDrawable interface.)
 
$repeat:Boolean (default = true) — If true, the bitmap image repeats in a tiled pattern. If false, the bitmap image does not repeat, and the edges of the bitmap are used for any fill area that extends beyond the bitmap.
 
$smooth:Boolean (default = true) — If false, upscaled bitmap images are rendered by using a nearest-neighbor algorithm and look pixelated. If true, upscaled bitmap images are rendered by using a bilinear algorithm. Rendering by using the nearest neighbor algorithm is usually faster.
 
$blendMode:String (default = null) — A string value, from the flash.display.BlendMode class, specifying the blend mode to be applied to the resulting bitmap.
 
$clipRect:Rectangle (default = null) — A Rectangle object that defines the area of the source object to draw. If you do not supply this value, no clipping occurs and the entire source object is drawn.
 
$rotation:Number (default = 0) — The rotation in degrees used by the Matrix for applying the fill.
 
$colorTransform:ColorTransform (default = null) — A ColorTransform object that you use to adjust the color values of the bitmap. If no object is supplied, the bitmap image's colors are not transformed.

Example
The following code creates a Bitmap Fill. 
    var bf :BitmapFill = new BitmapFill( new BitmapData( _loader.width, _loader.height, true ), _loader, true, true, BlendMode.OVERLAY, null, 0, null );
