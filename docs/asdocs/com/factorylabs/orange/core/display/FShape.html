public function FShape($container:DisplayObjectContainer = null, $init:Object = null)

FShape is the basic building block for display objects that normally utilize the Shape class.

Meant to be used in place of the native flash Shape class.

Parameters
$container:DisplayObjectContainer (default = null) — The DisplayObject to add this FShape to. If null this FShape will not automatically be added to the display tree of another DisplayObject.
 
$init:Object (default = null) — Object containting all parameters to automatically asign upon instantiation.

Example
The following code is the simplest example of how to use FShape. 
  // assumes holder is an existing display object
  // mc will be added to holder's display list and set to an x and y position of 10
  var mc:FShape = new FShape( _container, { x:10, y:10 } );
