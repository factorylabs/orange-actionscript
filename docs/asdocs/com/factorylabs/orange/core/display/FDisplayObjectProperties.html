Packagecom.factorylabs.orange.core.display
Classpublic class FDisplayObjectProperties

FDisplayObjectProperties allows you to gain access to additional properties of a DisplayObject.

Use this class sparingly as it adds a good amount of weight to a class. If only minor manipulation needs to occur, it is recommended to internalize some of the functions.

public function FDisplayObjectProperties($displayObject:DisplayObject, $bounds:Rectangle = null, $offset:Point = null)

Creates an instance for gaining additional access to properties of a DisplayObject.

Parameters
$displayObject:DisplayObject — The DisplayObject used to access additional properties.
 
$bounds:Rectangle (default = null) — A Rectangle used for size and coordinates â if null, the x, y, width and height properties of the $displayObject will be used.
 
$offset:Point (default = null) — A Point object used for offset properties â if null, a Point object will be created with 0, 0 properties.

Example
The following code creates a FDisplayObjectProperties classs. 
   var fsprite :FSprite = new FSprite( _sprite, { x: 0, y: 0 } );
   var rect :Rectangle = new Rectangle( 10, 10, 100, 100 );
   var offset :Point = new Point( 10, 10 );
   _fmock = new FDisplayObjectProperties( fsprite, rect, offset );
