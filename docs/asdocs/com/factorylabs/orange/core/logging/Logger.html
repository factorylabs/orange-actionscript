Packagecom.factorylabs.orange.core.logging
Classpublic class Logger
Implementsflash.events.IEventDispatcher, ILogger

Captures incoming messages and objects categorized by levels and a logger's identifier, broadcasts their log information out to observers.

Acts as a facade for capturing logging calls in an application and allows the results to be easily rewired to specific services. Logger's should be injected at startup.

Constructor detail
Logger()constructor
public function Logger($name:String = "[Generic Logger]", $isSilent:Boolean = false)

Instantiates a logger instance. Precondition: None. Postcondition: An object ready to be used for dispatching logging messages.

Parameters
$name:String (default = "[Generic Logger]") — A string identifier for the logger instance.
 
$isSilent:Boolean (default = false) — Whether this logger instance should suppress it's events.

Example
The following code instantiates a logger. 
    var logger :Logger = new Logger( '[Core Logger]', false );
    logger.trace( "[AbstractView].initialize()", _data, "MK" );

public function trace($msg:String = "", $object:Object = null, $level:String = null):void

Handles the basic necessities of logging a message, all class methods are filtered through here. Precondition: An optional message, optional object and optional level Postcondition: A dispatched log event if there are listeners and the logger isn't silenced.

Parameters
$msg:String (default = "") — String message to trace out.
 
$object:Object (default = null) — Object to be inspected via the debugger.
 
$level:String (default = null) — LogLevels constant or custom notification filter.
