core()method
public function core($msg:String = "", $object:Object = null):void

Logs the message and object through trace with the LogLevels.CORE filter.

Parameters
$msg:String (default = "") — String message to trace out.
 
$object:Object (default = null) — Object to be inspected via the debugger.

Example
The following code logs a command with a level of CORE. 
    var logger :Logger = new Logger();
    logger.level = LogLevels.CORE;
    logger.core( "[AbstractView].initialize()", _data );

trace()method 
public function trace($msg:String = "", $object:Object = null, $level:String = null):void

Handles the basic necessities of logging a message, all class methods are filtered through here. Precondition: An optional message, optional object and optional level Postcondition: A dispatched log event if there are listeners and the logger isn't silenced.

Parameters
$msg:String (default = "") — String message to trace out.
 
$object:Object (default = null) — Object to be inspected via the debugger.
 
$level:String (default = null) — LogLevels constant or custom notification filter.

Example
The following code logs a command with a custom level. 
    var logger :Logger = new Logger();
    logger.trace( "[AbstractView].initialize()", _data, "MK" );
