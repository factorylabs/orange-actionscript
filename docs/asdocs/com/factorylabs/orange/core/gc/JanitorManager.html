Packagecom.factorylabs.orange.core.gc
Classpublic class JanitorManager

Used for site wide clean up. This class is used primarily for cleaning up a .swf that gets loaded and disposed of so that all references can be cleaned out. Manages all of the Janitor instances so that clean up can be called on everything in the app.
