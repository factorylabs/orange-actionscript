Class JanitorTimerMethods
 
Packagecom.factorylabs.orange.core.gc
Classpublic class JanitorTimer
ImplementsIJanitor

The janitor is used for garbage collection dealing with Timer objects.

Janitor has methods within it to add and remove various objects that need to be removed from memory. There is also a one off clean method to collect everything.
